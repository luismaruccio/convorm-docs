# 🔌 Conexão

Atualmente o componente aceita duas formas de conexão:

1. Utilizando uma Connection String;
2. Passando os parâmetros ao criar o objeto de conexão;

### 📝 Utilizando uma Connection String

Para utilizar o arquivo de conexão basta criar um arquivo chamado *"connection.xml"* com as seguinte tags:
- "name": Nome da conexão;
- "driver" Driver a ser utilizado (Atualmente suporta-se os drivers "ecdtMySql" para MySql e MariaDB e "ecdtSQLServer" para SQLServer)
